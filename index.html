Ride distributor

Solution to the problem statement of the HashCode online quallification round 2018.

You can also check the output using this tool

Setup

  • Create Workspace ``` cmake . -B".build" -DCMAKE_EXPORT_COMPILE_COMMANDS=ON -DCMAKE_BUILD_TYPE=Debug ```
  • Create Executable (and documentation) ``` cmake –build .build –target all cmake –build .build –target doc ```
  • Unit Tests (first create executable) ``` cd .build && ctest -VV ```
